### headed / headless 이중 smoke

공개 `v0.1.3` asset 기준으로 `scripts/smoke-modes.sh`를 사용해 두 모드를 모두 검증했다.

- `headless`: `mode=headless`, `source=managed`, 클릭 후 `MODE-SMOKE-OK` 확인
- `headed`: `mode=headed`, `source=managed`, 클릭 후 `MODE-SMOKE-OK` 확인

주의:

- display가 없는 이 머신에서는 `Xvfb`를 띄워 headed smoke를 같은 DISPLAY 세션 안에서 실행했다.
- 각 명령을 별도 `xvfb-run`으로 감싸면 DISPLAY lifetime이 끊겨 headed smoke가 실패할 수 있다.

- `/workspace/agentab-cli/tmp/release-v0.1.3/session-start.json`
- `/workspace/agentab-cli/tmp/release-v0.1.3/doctor-before-stop.json`
- `/workspace/agentab-cli/tmp/release-v0.1.3/restart-daemon-status.json`
