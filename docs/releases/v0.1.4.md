# agentab-cli v0.1.4

`v0.1.3` 이후 누적된 운영성 개선과 액션 안정성 하드닝을 한 번에 묶은 patch release다.

## 포함 내용

- standalone `agentab-cli` cross-platform release artifact
- `doctor`, `daemon`, `session`, `tab` 명령 체계
- JSON-first CLI 계약
- PinchTab 기반 브라우저 런타임 관리
- screenshot / snapshot / pdf artifact 저장
- GitHub Actions + GoReleaser 기반 자동 release workflow

## 이번 릴리스에서 반영된 핵심 수정

- `doctor --output text`를 전용 운영 요약 형태로 정리
- `doctor`에 `logsDir`, `daemonLogPath`, `pinchtabLogPath` 노출 추가
- 관리형 artifact 응답에 `managed`, `relativePath`, `createdAt` 메타데이터 추가
- 명시적 잘못된 session / tab ID에 대한 `not_found` 오류 경로 테스트 보강
- `click`, `type`, `fill`, `press`, `scroll` 액션 smoke 확대
- PinchTab local runtime이 `${AGENTAB_HOME}`별로 가용 loopback 포트를 고르도록 하드닝
- 운영 런북과 릴리스 검증 이력 index 문서 추가
