# agentab-cli v0.1.2

`v0.1.1` 공개 릴리스 이후 발견된 CLI 사용성 문제를 보완한 patch release다.

## 포함 내용

- standalone `agentab-cli` cross-platform release artifact
- `doctor`, `daemon`, `session`, `tab` 명령 체계
- JSON-first CLI 계약
- PinchTab 기반 브라우저 런타임 관리
- screenshot / snapshot / pdf artifact 저장
- GitHub Actions + GoReleaser 기반 자동 release workflow

## 이번 릴리스에서 반영된 핵심 수정

- `tab open` 직후 `tab list`가 stale 목록을 반환할 수 있는 race 완화
- `tab list` 응답에 `currentTabId` 추가
- 현재 탭이 목록 맨 앞에 오도록 정렬
- `doctor`가 `CHROME_BIN` override를 `chromeBin`에 반영하도록 수정
- `doctor` 응답에 `chromeBinFound`, `chromeBinSource`, `chromeBinError` 추가
