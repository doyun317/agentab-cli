# agentab-cli v0.1.1

첫 공개 릴리스 후보를 실제 GitHub Release로 발행한 버전이다.

## 포함 내용

- standalone `agentab-cli` cross-platform release artifact
- `doctor`, `daemon`, `session`, `tab` 명령 체계
- JSON-first CLI 계약
- PinchTab 기반 브라우저 런타임 관리
- screenshot / snapshot / pdf artifact 저장
- GitHub Actions + GoReleaser 기반 자동 release workflow

## 이번 릴리스에서 반영된 핵심 수정

- `.gitignore`의 `agentab` 패턴이 `cmd/agentab` 디렉터리까지 무시하던 문제 수정
- `cmd/agentab/main.go`를 git에 포함해 CI release build가 정상 동작하도록 수정
- `AGENTAB_HOME`별 PinchTab runtime 격리 보완
