# agentab-cli v0.1.3

`v0.1.2` 이후 발견된 신규 환경 설치/종료 정리 이슈를 보완한 patch release다.

## 포함 내용

- standalone `agentab-cli` cross-platform release artifact
- `doctor`, `daemon`, `session`, `tab` 명령 체계
- JSON-first CLI 계약
- PinchTab 기반 브라우저 런타임 관리
- screenshot / snapshot / pdf artifact 저장
- GitHub Actions + GoReleaser 기반 자동 release workflow

## 이번 릴리스에서 반영된 핵심 수정

- `CHROME_BIN`과 PinchTab auto-install을 같이 쓸 때 config의 server port/bind/token 누락 수정
- 신규 환경 auto-install smoke 재검증
- daemon shutdown 시 daemon metadata뿐 아니라 pinchtab metadata와 session state도 같이 정리
- CLI 경로 기준 `lock_conflict`, `timeout`, `upstream_error`, 대표 `not_found` 종료 코드 검증 보강
